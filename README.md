# Biocinoz (C++ + SFML)

3D-модель замкнутой биологической системы:
- **Food (растения/корм)**;
- **Herbivore (травоядные)** едят корм;
- **Predator (хищники)** едят травоядных.

Реализованы:
- базовый класс `Entity` и наследники `Plant`, `Animal`, `Herbivore`, `Predator`;
- перегрузки операторов в `Vec3` (`+`, `-`, `*`, `+=`) и `operator<<`;
- 3D-пространство (аквариум/океан), визуализация через SFML (2D-проекция глубины);
- параметры жизни: возраст, голод, размножение, сексуальное/асексуальное поведение, скука, перенаселение.

## Сборка

```bash
cmake -S . -B build
cmake --build build -j
```

## Запуск

```bash
./build/biocinoz
```

Headless-режим (без окна):

```bash
./build/biocinoz --headless
```

Симуляция пытается прожить до **5000 итераций**.
Если любой вид вымирает (`food`, `herbivore` или `predator`), система считается погибшей.
